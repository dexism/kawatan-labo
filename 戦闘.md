# ルールセクション

## 戦闘

戦闘はバトルパートの大部分を占めています。
アンデッドたちの戦闘は人間のそれとは大きく異なり、腕がなくなろうと頭が取れようと、まだまだ続きます。NCもPLも、安心して破壊し合いましょう。
戦闘はドールとNCが操る手駒の間で発生します。

### 戦闘の準備

戦闘は、「舞台見取図」と呼ばれる記号化された地図上で行われます。地図は5つの領域に分かれています。それぞれの領域を「エリア」と呼びます。
PLは、作成時に決めておいたドールの「配置」エリアに自身を示すマーカーを置いてください。
続いて、NCはこの戦闘における「勝利条件」をPLに示します。この際、追加のカルマ(後述)があれば、それも宣言しましょう。その後、用意した手駒を示すマーカーを任意のエリアに配置します(配置エリアは予め決定しておきましょう)。
さらに「舞台見取図」の右側にある「15」～「-5」の数値が書かれたマスにもマーカーを配置します。PLとNCは戦闘参加者それぞれの最大行動値と同じマスへ、マーカーを配置してください。
以上の準備が終了したなら、PLとNCは「バトルパート開始時」に処理すべきマニューバなどを待っていないか確認しましょう。
何もないか、処理が終了したなら、戦闘を開始します。

### 勝利条件と全滅

戦闘には様々な状況があり、状況によってNCから異なる勝利条件が与えられます。
ドール側が勝利条件を満たしたなら、ドールたちの勝利となり、手駒が未だ残っていてもバトルパートは終了します。残った手駒らは、その場の指揮官を失って無意味な行動しか取れなくなったと考えてください。
逆に姉妹全員が攻撃手段・移動手段の両方を失うか、全ての未練を狂気で埋め尽くされてしまったなら、その時点でドールたちは全滅したものと見なされます。
バトルパートは終了し、修復はできません。エンドパートの『エンディング』(P96)に進んでください。
全滅したドールに活躍の機会はもうありません。ただどのように処分されたかだけ、NCから語って終わりとしてください。全滅した際も、相応の龍愛点を獲得することはできます。全滅したドールは他のセッションにも参加できません。運がよければ部品くらいは、次のドールの材料に活用されるかもしれませんが……。

### 戦闘の進行

戦闘は、ターンと呼ばれる時間単位で区切られています。1ターンはゲーム内の時間でおよそ10秒です。
ターンはさらにカウントと呼ばれる時間単位で区切られています。カウントは戦闘参加者の行動値と密接に関わっています。
カウントは、戦闘参加者の中で、最も高い最大行動値を基準に開始されます。この数値から0へと、NCの宣言によって1つずつ減っていきます。舞台見取図の右に書かれた数値にそれぞれマーカーを置き、行動するカウントを管理してください。行動値が変化する都度、マーカーを動かして現行動値について他にわかるようにしましょう。
戦闘参加者は全て、最初のターンにおいて、最大行動値と同じ行動値を持っています。カウントは戦闘参加者が行動していくとで減少します。

カウントが「0」になったら、ターンは終了です。この時、全てのPLは任意の未練を選び、そこに狂気点を1点加えなければいけません。

次ターンの開始時、戦闘参加者全員の行動値が最大行動値分だけ回復します。この際、「最大行動値に回復する」わけでないことに注意ください。
この時、最大行動値を高めていたパーツが損傷しているかもしれません。その場合、最大行動値が減少するため、回復する行動値も変わります。
例えば、最大行動値10のドールが行動値マイナス2でターンを終えたなら、10点回復して行動値は8点です。もし【のうみそ】や【めだま】を損傷していたら、回復は10点より低くなり、行動値も7以下から始めることとなるでしょう。

戦闘参加者全てが行動値を回復させ、まだどちらも敗北していなないなら、新しいターンのカウント処理を開始します。
どちらかが敗北するか逃走するまで、以上の手順を繰り返してください。

### 戦闘の手順

1.  **バトルパート開始時**
    *   勝利条件の宣言
    *   カルマの宣言
    *   マーカーの配置

2.  **ターンの開始**
    *   現行動値を最大行動値分だけ回復する
    *   カウントダウンを開始する

3.  **カウントの処理**
    *   現行動値がカウントと同じキャラクターが行動する
    *   該当キャラクターがいない場合、カウントを減らす
    *   カウントが0になった → 4へ
    *   ドールたちが全滅した → 5へ
    *   勝利条件を満たした、または逃走に成功した → 5へ

4.  **ターンの終了**
    *   任意の未練に狂気点を1点加える
    *   新たなターンを開始する → 2へ

5.  **バトルパート終了**
    *   全滅の場合：エンドパートのエンディングを開始する
    *   それ以外の場合：「たからもの」への狂気点を回復させる
    *   バトルパートを終了し、エンドパートを開始する

### 行動

カウントが自身の現行動値と同じになった戦闘参加者は、所有する「アクション」タイミングのマニューバを使用できます。
あるいは、マニューバを使用せず、待機を宣言してもかまいません。待機したなら、行動値は1点減少します。
使用するマニューバを宣言したなら、そのコスト数値分だけ、行動値が一時的に減少します。
カウントが減って、減少した行動値と同じ数値になったら、再び「アクション」タイミングのマニューバを使用できます。ただし、各マニューバは、受けたダメージによって使えなくなってしまうこともあります。注意しましょう。

「ジャッジ」「ダメージ」「ラピッド」タイミングのマニューバは、行動値のカウントに関係なく、タイミング条件に一致していれば宣言し、使用できます(ラピッドはアクションと同様にも使用できます)。ただし、これらのタイミングのマニューバは、それぞれ1ターン1回ずつしか使えません。

コストを支払えば行動値がゼロ以下となる場合も、マニューバを使用することはできます。その際、行動値はマイナスとなることに注意ください。なお、すでに行動値が0以下の場合、「オート」以外のマニューバは使用できません。
例えば、カウント2でコスト4のマニューバを使用した場合、行動値はマイナス2となります。行動値がゼロ以下となった参加者は、次のターンになるまで、「オート」のタイミングのマニューバ以外を使用できません。

### カウント内の処理

同じカウントで、現行動値が同じになった敵味方が入り乱れて行動することも、戦闘では多々発生するでしょう。
こうした同カウントの「アクション」は、全て同時に起きるものと扱います。
例えば、移動マニューバを使った対象に攻撃マニューバを使用する際、対象はまだ移動前のエリアにいると考えます。ただし、「ラピッド」「ジャッジ」のタイミングで移動したなら、移動は攻撃より先に行われるのです。

同カウント内の行動処理は、必ずNC側から行ってください。このNC側の攻撃でパーツが損傷しても、ドールたちは同カウント内で損傷したパーツのマニューバを使用できます(コストなどには使用できません)。
ただし、「ラピッド」「ジャッジ」タイミングのマニューバはこうした同カウント内の同時行動にも割り込んで先に処理されます。「ラピッド」や「ジャッジ」で攻撃され、損傷したパーツのマニューバは、同カウント内でも使用できません。
同時に発生するため、「ダメージ」「ジャッジ」「ラピッド」タイミングでマニューバを使用し、行動値が減少したとしても、そのカウントにおいて「アクション」タイミングのマニューバを使用できます。ただし、現行動値が0以下となってしまったなら、通常と同じくあらゆるマニューバは使用できません。

同タイミングでも、「ダメージ」「ジャッジ」「ラピッド」のタイミングのものは、一体の戦闘参加者が複数を発動させることができます。極端な例を挙げれば、ある判定に対して一人のドールが【うで】で支援しつつ【あし】で妨害することも可能です（意味はありませんが）。
同様に、複数の戦闘参加者が1つのアクションマニューバに対して「ダメージ」「ジャッジ」「ラピッド」マニューバを発動させることも可能です。
なお、こうしたタイミングが重なり合った際は、後から発動させたマニューバを優先して処理します。「ラピッド」マニューバをさらに「ラピッド」で阻止したり無効にしたりすることも可能です。複数の戦闘参加者が入り乱れた際には、「ラピッド」「ジャッジ」「ダメージ」のタイミングについては、後から出されたものを優先して処理してください。

### カウントの処理

1.  **行動できるキャラクターを確認する**

2.  **「アクション」のマニューバを宣言する**
    * 行動値が同じキャラクターがいた場合、NC側が先に行動宣言を行う。それ以外は任意。
    * 「ラピッド」のマニューバを「アクション」扱いで宣言することも可能。
    * 宣言と同時に行動値を減少させる。

3.  **判定を行う**
    * まだ行動できる → 2へ
    * 全員の行動値がカウント未満 → 4へ

4.  **マニューバの効果を適用する**
    * 「アクション」のマニューバの効果を適用する。
    * 効果は同時に適用され、互いに影響は与えない。

5.  **カウントの減少**
    * ネクロマンサーはカウントの減少を宣言し、次のカウントの処理を行う。

### 「アクション」以外のマニューバ

「アクション」タイミングでのマニューバの効果は、カウント減少前に同時に与えられるが、以下の「アクション」タイミングでないマニューバの場合、効果はあとに宣言されたほうから順に適用する。その結果、パーツの破損、対象の不在などでマニューバの効果が発揮できなくなった場合、この行動は自動的に失敗となる。

### 「アクション」以外のマニューバの宣言タイミング

*   **ラピッド**
    他のマニューバが宣言された際に宣言できる。

*   **ジャッジ**
    射程内で判定が行われた直後、効果が適用される前に宣言できる。

*   **ダメージ**
    射程内の対象にダメージが発生した直後、ダメージが適用される前に宣言できる。

*   **オート**
    常に効果を発揮する。宣言の必要はない。

### 攻撃判定

効果が「攻撃」のマニューバを使用した際に行います。判定し、攻撃判定表で命中箇所を決定し、判定値に応じた結果を得ます。

基本的には、判定値6以上が出たなら、数値に応じた箇所へ、マニューバに応じたダメージを与えます。
大失敗なら、攻撃は射程エリア内の味方か攻撃者自身に命中したものと扱います（全体攻撃の効果があっても、それは失われます）。
大成功なら、攻撃者は任意の箇所を選んで命中させ、与えるダメージを［判定値-10］点だけ上乗せさせます。

| 02     | 失敗                                                                                 |
| 03     | 失敗                                                                                 |
| 04     | 失敗                                                                                 |
| 05     | 失敗                                                                                 |

### ダメージ

攻撃などによって発生したダメージは、頭・腕・胴・脚のいずれかの箇所に与えられます。ダメージを受けた箇所のパーツを、ダメージと同じ数だけ損傷させてください。
損傷したパーツはエンドパートに修復(後述)するまで、効果が失われます。マニューバとして使用することはできません。既に損傷しているパーツをさらに損傷させることもできません。
命中した箇所のパーツがすべて損傷しているなら、攻撃者は残された中から任意の箇所を選ぶことができます。
全ての箇所、全てのパーツが損傷してしまったドールは、修復されるまで戦闘中にできることはありません。

### 切断判定

体を一刀両断する攻撃を受けたら、たとえ動けるパーツが残っていても一撃で体の一部が切り落とされるかもしれません。
こうしたことを表すため、「切断」効果のある攻撃を受けた対象は判定を行います。
この判定に失敗したなら、その箇所の全てのパーツはダメージに関係なく、全て損傷してしまいます。

### 逃走

どうしても戦闘で勝てそうになく、NCがそれを認める状況でなら、逃走を試みることもできます。
これは「楽園」にいるドールのみ、試みることができます(サヴァントの場合は逆に「奈落」から逃走します)。全員が移動マニューバを使用し、行動判定と同様に判定してください。
この判定は戦闘中のもので、ジャッジタイミングのマニューバを行うこともできます(距離は「楽園」から1離れているものとします)。
成功したドールは戦闘から無事に逃げ出せたこととなります。
失敗したなら、戦場に残らなくてはいけません。「楽園」に配置されたままです。移動マニューバを使い続ければ、何度でも逃走を試みることができます。
もし大失敗なら、ターンが変わるまで、再度の逃走を試みることはできません。
なお、全てのドールが逃走を行うか、残ったドールが全滅した場合、エンドパートでの修復(後述)ができません。倒した敵や自身の損傷した部分を、戦場に残していくのですから。損傷によっては、次のセッションがなお過酷なものとなることでしょう。

### その他、発生しうること

バトルパートでも、他のルールを使用すべき状況が発生するかもしれません。

例えば、行動判定を行わなければならない状況もありえます。
そうした際は、基本的にアクションタイミングで、NCの裁定したコストを消費して判定することになるでしょう。以下にいくつかの例を挙げておきます。

#### 戦闘中の行動例

**何かに気づく**

ラピッドタイミングでコスト0として行ってよい。ただし、注意して観察しなければいけないなら、アクションタイミングでコスト1として行う。

**書面を読み解く**

難解さや分量、あるいは重要度にもよるが、それを行うドールはアクションタイミングで最低コスト1を支払う。

**扉を開く**

移動マニューバと同コストとする。あるいは開錠やパスワード入力が必要なら、コスト3～4程度。

**壁を破壊する**

攻撃して一定ダメージで破壊される敵とて配置する。

**機械を稼動させる**

戦況を大きく変えるものなら、コスト3～4程度。軽微なものや複数回の稼動が必要なら1～2程度でもいいだろう。

**壁や崖を登る**

移動マニューバと同コストとする。角度によって修正が発生し、【かぎづめ】などが有利に働くだろう。

**重いものを運ぶ**

基本的に【こぶし】や【かた】のコストをそのまま流用。ドールは筋力が高く、【怪力】などのスキルがあればさらに強化される。

なお、戦闘中でも敵のマニューバや、状況によって任意判定は発生します。詳しくはネクロマンサーセクションを参照してください。
ただし、対話判定は何らかのマニューバを使わない限り、バトルパート内では発生しません。それ自体が狂気を高める戦闘中に、対話で平静を取り戻すことは容易ではないのです。

### 戦闘後の修復

戦闘やトラブルによって、ドールが体を損傷することは多々あります。
しかし、ドールたちはアンデッドです。それゆえ、他者のものでも使えそうなパーツを手に入れれば、体を修復することができます。縫い合わせたり、接着したり、再生能力を働かせたりして体をつなぐのです。

バトルパートが終わり、エンドパートの開始時に修復は行われます。戦闘で勝利しなければ、修復は行えません。
戦闘後に修復できるパーツ数は、敵の悪意の合計によって変わります。勝利さえしていれば、逃走した敵や破壊せずに戦闘終了した敵がいても、合計悪意点は変わりません。
以下の式に当てはめ、姉妹全体の修復可能なパーツ数を求めてください。

**基本パーツ数 ＝ (敵の合計悪意点) ÷ 4 (切上げ)**
**強化パーツ数 ＝ (敵の合計悪意点) ÷ 8 (切上げ)**

得られたパーツは、姉妹全体の共有ストックです。それぞれに必要なパーツを宣言し、分け合うようにしましょう。

基本パーツは比較的簡単に修復できますが、強化パーツはなかなか修理できません。損傷させる際にも、そうしたことは念頭に置いた方がいいでしょう。
また、強化パーツを回復する際は、本来と別の強化パーツ(ただし同じ強化タイプで、同じかそれ以下のレベルに限ります)に交換してもかまいません。この際、同じパーツを複数獲得することはできません。
もし強化パーツに誰もダメージを受けておらず、基本パーツばかりが損傷しているなら、修復できる強化パーツ数を減らし、その数だけ基本パーツを修復してもかまいません。ただし、基本パーツが強化パーツの代わりになることは決してありません。
最後に、重要な点として損傷した「たからもの」を修復することはできません。新たに手に入れるしかないのです。

### カルマ

「カルマ」は、ドールたちが背負う目的や使命です。カルマはたいてい複数が与えられます。内容がドールごとに異なることもあるでしょう。基本的に、ドールらが1回のセッションで背負う「カルマ」は2～3個です。
全てのドールは初期から『記憶のカケラを獲得する』というカルマを持っています。このカルマは、セッション内で「記憶のカケラ」を1つ以上獲得したなら(得た記憶のカケラ1つごとに、ではありません)、達成したと見なされます。
カルマは「アドベンチャーパート開始時」か「バトルパート開始時」、NCからPLに提示されます。こうしたカルマの達成条件は内容によって異なるでしょう。PLは、NCにカルマの達成条件について疑問があればいつでも質問してかまいません。
セッション終了時、達成したカルマ1つごとに龍愛点を2点追加で獲得できます。
カルマを達成することは、ドールの強化を促進するため重要な指針となるでしょう。

### 龍愛

この世界で正気と肉体を保ち続けるドールには、ネクロマンサーから歪んだ「龍愛」がもたらされます。
龍愛はドールの体を歪め、新たな能力を与えるでしょう。
セッション終了時、ないしセッション開始前に龍愛を消費することで、ドールは表の効果のいずれかを得ることができます。

| 消費龍愛点 | 効果                                                                                                              |
| :-----: | :------------------------------------------------------------------------------------- |
| 1点  | 未練の内容1つを任意のものに変更する。ただし、「たからもの」への未練は変更不可。           |
| 2点  | 「他の姉妹」「たからもの」以外の未練1つを削除する。              |

| 10点 | 望む強化値を＋1にする (それぞれ最大9まで)。増えた強化値に応じた強化パーツ(ドールセクション参照)を1つ、新たに獲得する。 |
| 20点 | 自身と異なるポジションもしくはクラスの未習得スキルの中から、1つ選んで習得する (特化スキルは不可)。 |

新たな未練を獲得できるのは、セッションで未練の対象たりうる存在が現れた時だけです。ドールたちに新たな仲間が増えたり、何度も登場させたいサヴァントやネクロマンサー自身と深く関わったなら、NCは彼らとの未練を作ることができると、龍愛を渡した際に宣言しましょう。
この時、未練の内容はNCの許可を得れば任意のものとしてかまいません (また、NCから特定の内容を推奨してもいいでしょう)。新しい未練は狂気点が3点溜まった状態から始まります。
新しい未練は、次回以後のセッションでより強い心を持つ役立つでしょう。
ただし、未練は最大でも6種類しか獲得できません。ご注意ください。

龍愛はあくまでドール個人に与えられるものですが、全員が「精神崩壊」したり、戦闘で全滅してしまった場合、龍愛点は次にそのPLが次に造るドールに与えられます。前回の失敗を見て、ネクロマンサーはより強力なドールを造る、というわけです。
また、セッションの形式によっては、NCも龍愛点を獲得できる可能性があります。詳細はネクロマンサーセクションを参照ください。
